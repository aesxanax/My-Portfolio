Информация о HTML5 и CSS3

HTML

- язык гипертекстовой разметки. Нужен для создания макета страницы

CSS

- каскадные таблицы стилей. Нужны для форматирования страницы

Благодаря HTML и CSS мы можем создать и красиво отформатировать страницу
